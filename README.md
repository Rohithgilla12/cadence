# Cadence

> The quiet rhythm of becoming.

A habit tracker with context-aware analytics. Built first for a small group of runner friends.

## This repo

Single git repo containing both apps:

- **`cadence-mobile/`** — Expo + React Native + TypeScript
- **`cadence-api/`** — Go service + Postgres

Source of truth for product, design, and conventions:

1. [`docs/PRD.md`](docs/PRD.md) — what we're building and why
2. [`docs/DESIGN_SYSTEM.md`](docs/DESIGN_SYSTEM.md) — visual language, components, tokens
3. [`CLAUDE.md`](CLAUDE.md) — coding conventions
